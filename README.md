> 資料來源：內政部不動產實價登錄 | 共 44,979 筆預售屋交易資料

## 🌐 系統網址

**https://ellen3149-mogo.github.io/HTML/**

## 📁 檔案說明

| 檔案 | 說明 |
|------|------|
| `index.html` | 主系統（整合四大功能於單一頁面） |
| `data.json` | 所有交易資料（44,979 筆，~11MB） |
| `staticwebapp.config.json` | Azure Static Web Apps 設定檔（備用） |

---

## 🧩 四大功能模組

### 🔍 搜尋篩選
- 依行政區、建物型態、房間數、電梯、總價區間、單價區間多條件篩選
- 建案名稱關鍵字模糊搜尋
- 點擊欄位標題排序（總價、單價、坪數等）
- 分頁瀏覽（每頁 20 / 50 / 100 筆）
- 點擊「詳情」查看完整交易資訊

### 📊 統計分析
- 各行政區平均單價長條圖
- 各區成交量環形佔比圖
- 建物型態分布橫條圖
- 總價區間分布直方圖
- 可依行政區篩選格局分布

### 🧮 價格計算
- 由單價推算總價（含頭期款、貸款、月繳試算）
- 由總價反推每坪單價
- 薪資負擔率分析（健康 ≤30%、偏高 30-40%、過高 >40%）
- 各行政區均價參考表（含最低、最高、平均）

### 📍 周邊生活機能
- 輸入地址自動定位（OpenStreetMap）
- 查詢周邊 500m ～ 2km 範圍內設施
- 支援六大類型：🏥 醫院診所、🎓 學校、🛒 超市賣場、🚇 捷運站、🌳 公園綠地、🏦 銀行
- 互動地圖標示，點擊清單自動飛到地圖位置
- 結果依距離排序並顯示精確公尺距離
- 完全免費，使用 OpenStreetMap + Overpass API，無需 API Key
